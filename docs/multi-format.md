# Multi-format emission — `euxis_publisher.emit`

Sprint 6 (S6.2 + S6.3) ships HTML5 and JATS XML emitters that
convert a built LaTeX document into single-source multi-format output.
Closes Forcing Function #3 in
[`strategy-2026.md`](strategy-2026.md): single-source PDF + HTML + JATS
(EPUB-A coming in S6.6).

Both emitters wrap Pandoc — the engine doesn't reimplement a LaTeX→X
converter, just composes the right `pandoc` flags and post-processes
the output for accessibility hints Pandoc doesn't emit.

## Install

Pandoc is a runtime dependency:

```bash
brew install pandoc          # macOS
apt install pandoc           # Debian / Ubuntu
nix-env -iA nixpkgs.pandoc   # Nix
```

The Python wrapper itself ships with the engine — no extra pip install
needed.

## API

```python
from euxis_publisher.emit import pandoc

# HTML5 (WCAG 2.2 AA target via semantic markup + skip-link)
result = pandoc.emit_html(
    tex_path=Path("src/papers/whisper.tex"),
    output_dir=Path("build/papers"),
    doc_id="whisper-paper",
    title="Real-Time ASR with Whisper",   # optional override
    lang="en-GB",                          # BCP-47 language code
)
print(result.output_path)   # build/papers/whisper-paper.html
print(result.bytes)         # size of generated artefact

# JATS XML 1.3 (archiving DTD; Crossref / PMC / JATS4R compatible)
result = pandoc.emit_jats(
    tex_path=Path("src/papers/whisper.tex"),
    output_dir=Path("build/papers"),
    doc_id="whisper-paper",
    title="Real-Time ASR with Whisper",
)
print(result.output_path)   # build/papers/whisper-paper.xml

# Both at once
results = pandoc.emit_all(
    tex_path=Path("src/papers/whisper.tex"),
    output_dir=Path("build/papers"),
    doc_id="whisper-paper",
    formats=["html", "jats"],   # default: both
)
```

## HTML5 — accessibility specifics

Pandoc's `--to html5 --standalone --section-divs` emits close-to-clean
WCAG 2.2 AA output. The emitter post-processes the result to add:

| Injection | Why |
|---|---|
| `<a href="#main" class="skip-link">Skip to main content</a>` | WCAG 2.4.1 (bypass blocks). Off-screen by default, focus-visible on tab. |
| `<!-- Generated by Euxis Publisher … -->` provenance comment | Supply-chain audit + reader-tooling identification. |
| `--metadata=lang:<bcp47>` → `<html lang>` | WCAG 3.1.1 (language of page). Default `en-GB`. |

Post-processing is **idempotent** — running it twice on the same file
produces identical bytes, so the HTML can be regenerated in CI without
diff churn.

Known limitations (Sprint 6 scope):

- Pandoc does not auto-tag decorative figures as `role="presentation"`.
  Manual review for figure captions still required for axe-core
  cleanliness.
- Math is emitted as MathJax (`--mathjax`). For native MathML, swap
  to `--mathml` when targeting accessibility-strict readers.
- No automatic table-of-contents `role="doc-toc"` injection yet —
  Sprint 7 work.

## JATS XML — what you get

`emit_jats` uses Pandoc's `jats_archiving` writer (JATS 1.3,
archiving-and-interchange DTD). The output is accepted by:

- Crossref deposit (with a Crossref-specific schema overlay)
- PMC (PubMed Central)
- bioRxiv / medRxiv
- arXiv (as supplementary metadata)
- JATS4R recommendations (passes most rules; full JATS4R compliance
  requires a separate validator step — not bundled here)

The emitter does NOT include reference resolution — `\cite` entries
become `<xref>` placeholders that need a bibliography back-end
(Crossref REST API or `jats4r-bib`). This is Sprint 7 work.

## CLI

Sprint 7 wired the emitters into the build CLI:

```bash
# Emit both formats for every registered document
python -m euxis_publisher.cli.build emit
# or via Make:
make emit

# Single document, single format
python -m euxis_publisher.cli.build emit --doc whisper-paper --formats html
make emit-html

# Strict mode for CI — exits 1 on any pandoc failure
python -m euxis_publisher.cli.build emit --strict
```

The registry filter from `cli.audit` applies — only documents listed
in `data/meta.yaml documents:` get emitted, and entries flagged with
`note: This is an input file used by another` are skipped.

JATS4R validation + the EPUB-A emitter (S6.6) land in Sprint 7.5;
track [docs/audit-2026-05.md](audit-2026-05.md) §5.
